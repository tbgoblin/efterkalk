# Brugermanual (kort)

## 1. Dashboard
- Viser makrokategorierne Salg, Produktion og HR.
- Brug kortene til at aabne moduler.
- Warmup-status viser baggrundsforberedelse af cache.

## 2. Efterkalkulation
- Viser ordreliste, margin, kost og detaljer pr. ordre.
- Klik paa en ordrelinje for rapportvisning.
- Knappen `Opdater` paa en ordre rydder cache for den enkelte ordre og henter nye beregninger.

## 3. Omsaetning
- Analyse af omsaetning pr. periode, konto og kunde.
- Vaelg filtre og tryk `Opdater`.
- Brug print-knappen efter opdatering for korrekt rapport.

## 4. Ordreindgang
- Analyse af ordre/tilbud pr. ugeinterval.
- Vaelg `Fra uge` og `Til uge` (YYYYWW), og tryk `Opdater`.

## 5. Datadifferencer (NestKost)
- `NestKost pr. stk` beregnes fra linjens samlede kost og det anvendte linjeantal.
- Routedetaljer kan vise andet antal, hvis nesting-fordelingen har ekstra/manglende stk paa ruter.
- Ved mismatch vises markering i UI.

## 6. Praktisk workflow
1. Vaelg modul.
2. Saet filtre.
3. Tryk `Opdater`.
4. Tjek KPI + tabel.
5. Print/eksporter ved behov.
