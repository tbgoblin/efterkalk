How it works:
1. User builds new version: npm version patch && npm run build:win
2. Developer creates GitHub Release with new .exe
3. Running app checks GitHub every startup
4. If new version available, downloads in background
5. Notifies user with dialog: "Opdatering tilgængelig" 
6. User can restart now or later to apply update
